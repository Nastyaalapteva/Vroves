streamALL


An engine which allows you to push single stream at one time to multiple services like Facebook, YouTube, ustream, etc...
SDK contains of engine and backend for the server which will do transcoding and deploy to the services. That approach allows user to post a stream to multiple locations within a single click. Server side will make a decision on required transcodings or simply redirect stream to the required service.
